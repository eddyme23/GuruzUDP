# GuruzUDP
UDP INSTALLER FOR HYSTERIA 1, HTTP CUSTOM &amp;  ZiVPN

UDP server installation for Hysteria 1, Http Custom and ZiVPN Tunnel.
<br>

#### Installation
```
wget -O zi.sh https://raw.githubusercontent.com/zahidbd2/udp-zivpn/main/zi.sh; sudo chmod +x zi.sh; sudo ./zi.sh
```
